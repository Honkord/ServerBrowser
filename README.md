Server Browser is an indie local proxy server designed for remote
web browsing. The idea of this project is to create a private 
server that mimics like a typical browser to bypass websites 
through security programs, such as GoGuardian. 

Disappointedly, the project has came to result that I have
inadvertently wasted my time on this project because GoGuardian
blocks the proxy server nevertheless saddly after all.
